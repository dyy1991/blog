# Hello World

每个程序员的第一行代码都是 `Hello World`，这里也不例外。

## 为什么开始写博客

写博客的原因很简单：**记录、整理、分享**。

学习一个新技术的时候，最好的方式是能够用自己的语言把它讲清楚。如果讲不清楚，说明还没真正理解。这个博客就是这样一个地方——把脑子里散乱的知识整理成能让别人（和未来的自己）看懂的文章。

## 这里会写什么

- **技术笔记**：CI/CD、Docker、云原生、AI 工程化
- **项目记录**：从想法到落地的完整过程
- **踩坑备忘**：那些花了好几个小时才搞定的 bug
- **随想**：关于编程、学习、工具的碎碎念

## 这个博客本身

这个博客用 Next.js 构建，部署在阿里云 ECS 上，通过 GitHub Actions 实现 CI/CD 自动部署。每次 `git push`，代码自动构建、测试、部署——这也是我最近在学习的方向。

代码开源在 GitHub，欢迎围观。

```bash
# 本地开发
npm run dev

# 提交自动触发部署
git push origin main
```

## 开始了

就这样，开始了。
